# Contributing to this monitor

This repository is a **catalog and personal monitor** for upstream PyPTO-related projects. It is **not** the place to request changes to those upstream repositories.

If you want to change behavior, fix bugs, or propose features for a listed project, open issues or pull requests **on that upstream repository** (and follow its contribution guidelines).

Changes welcome **here** only for: improving the manifest, README, automation that stays read-only toward upstream, or documentation about how you track releases.
